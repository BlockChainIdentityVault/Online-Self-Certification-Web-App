Getting Started

To complete the OpenChain Conformance Check, applicants must provide a self-confirmation. The Conformance Check requires you to:
  • Fulfill all requirements set forth in the Specification [LINK to Specification].
  • Create an account by selecting sign-up in the user menu.
  • Name a contact person to handle inquiries.
  • Have an authorized representative complete the web-based Conformance Check by answering all the questions.
  • Fill out the submission form.
  • Archive a Reference Copy of all Verification Artifacts as a record in support of the Conformance Check.
